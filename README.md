# The real course-management

## Gettin' started

* install dependencies

  ```bash
  bower install
  python3 -m pip install -r requirements.txt
  ```

* Initialize the database

  ```bash
  ./init.sh
  ```

* Start the server

  ```bash
  python3 manage.py runserver
  ```

* You now can login with the super user **foo** and password **bar**

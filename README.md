# The real course-management

## Gettin' started

 * install requirements

```bash
python3 -m pip install -r requirements.txt
```

 * go to folder *src*
 * Initialize the database
```bash
 python3 manage.py migrate
```

 * Add a superuser
```bash
 python3 manage.py createsuperuser
```

 * Start the server
```bash
 python3 manage.py runserver
```
